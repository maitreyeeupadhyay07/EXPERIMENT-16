
## **Aim**

To study and implement basic **Natural Language Processing (NLP) techniques** in Python using the NLTK library, including tokenization, stop word removal, stemming, lemmatization, part-of-speech tagging, and word frequency analysis.

---

## **Theory**

**Natural Language Processing (NLP)** is a branch of artificial intelligence that deals with the interaction between computers and human language. It enables machines to understand, interpret, and process textual data.

In this experiment, several fundamental NLP techniques are performed using the **NLTK (Natural Language Toolkit)** library:

### **1. Tokenization**

Tokenization is the process of breaking text into smaller units such as words or sentences.

* **Word Tokenization** splits text into individual words.
* **Sentence Tokenization** divides text into sentences.
  This is the first step in text processing.

### **2. Stop Word Removal**

Stop words are common words like *is, the, in, and* which do not carry significant meaning.
Removing them helps in focusing on important words and improves processing efficiency.

### **3. Stemming**

Stemming reduces words to their root or base form by removing suffixes.
Example: *running → run*
It is fast but may not always produce meaningful words.

### **4. Lemmatization**

Lemmatization converts words into their correct dictionary form (lemma).
Example: *studies → study*
It is more accurate than stemming because it considers grammar and context.

### **5. Part-of-Speech (POS) Tagging**

POS tagging assigns grammatical labels such as noun, verb, adjective, etc., to each word.
Example: *Python (Noun), is (Verb)*
This helps in understanding sentence structure.

### **6. Word Frequency Count**

This technique counts how often each word appears in a text using tools like **FreqDist**.
It is useful for text analysis, keyword extraction, and identifying important terms.

---

## **Conclusion**

In this experiment, various NLP techniques were successfully implemented using Python and the NLTK library. The experiment demonstrated how raw text can be processed and analyzed through tokenization, stop word removal, stemming, lemmatization, POS tagging, and frequency distribution. These techniques form the foundation of advanced applications such as chatbots, sentiment analysis, and text classification.
